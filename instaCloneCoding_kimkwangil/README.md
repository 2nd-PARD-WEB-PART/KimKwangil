# 인스타그램 클론 코딩

- 파드 세미나 과제로 진행되는 프로젝트로, 인스타그램을 직접 클론 코딩하여 구현하는 것으로 목표하고 있다.

- 사용 언어 : javascript
- 라이브러리 : React
- 개발 환경 : vscode
- 개발 인원 : 1명 (나홀로 개발)
- 개발 출처 : Pard 2기 웹 파트
