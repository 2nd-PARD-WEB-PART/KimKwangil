## 짧은 회고

<p><label>5차 과제 중 남기는 회고 231103 02:28</label><br><br>사실 인스타그램 클론 코딩을 잘 따라갈 수 있을까 싶은 생각도 들었다. 그래도 막상 해보니까 배운 내용들을 적용시켜 나가는 부분이 재밌었고, 시간 투자하는 것이 아깝지 않게 느껴진 것 같다. 세미나 때 새로 배우는 내용이 많아질수록 더더욱 재밌게 프로그래밍할 수 있었다. 4차 과제까지는 late된 점도 많고, 부랴부랴 기능을 구현하느라 바빴다면, 5차 과제에는 배운 점을 적용시키는 점도 기간 안에 끝낼 수 있었고, 무엇보다 주석이나 깃허브 관리까지 할 수 있던 점이 가장 마음에 든 순간이었다. 프로젝트를 혼자 하더라도 이렇게 README 파일을 작성하고, repo 관리까지 한다는 점이 얼마나 중요한지 많이 배울 수 있었으며, 이 과정을 통해서 앞으로의 1인 개발 혹은 다인 개발을 할 때에 반영을 많이 할 수 있을 것 같다. 초기에 용현님이 하신 ot homework의 Repo를 본 적이 있는데, 당시에 이미 readme 파일과 폴더 정리 및 주석까지 달아 놓으신걸 보면서 감탄을 안 할 수가 없었다. 아무튼 정말 이 프로젝트를 하면서 많이 배워간 것 같다.</p>
